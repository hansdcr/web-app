### 认证流程

```
注册流程:
用户填写表单 → 表单验证 → 调用注册API →
自动登录 → 保存token → 跳转到首页

登录流程:
用户填写表单 → 表单验证 → 调用登录API →
保存token → 更新认证状态 → 跳转到首页

退出流程:
点击退出 → 调用退出API → 清除token →
清除认证状态 → 跳转到登录页

路由守卫:
访问受保护页面 → 检查token →
有token: 允许访问
无token: 跳转登录页
```

### API接口

基于 cultrue 项目的API：

1. **注册**: `POST /api/auth/register`
   ```json
   {
     "username": "string",
     "email": "string",
     "password": "string"
   }
   ```

2. **登录**: `POST /api/auth/login`
   ```json
   {
     "email": "string",
     "password": "string"
   }
   ```

3. **退出**: `POST /api/auth/logout`
   ```json
   {
     "refresh_token": "string"
   }
   ```

4. **获取用户列表**: `GET /api/users`
   - 需要认证（Bearer token）

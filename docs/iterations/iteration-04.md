## 📊 页面设计

### 添加用户表单

```
┌─────────────────────────────────────────┐
│  添加用户                                │
├─────────────────────────────────────────┤
│                                          │
│  用户名：[_____________] *               │
│                                          │
│  邮箱：  [_____________] *               │
│                                          │
│  密码：  [_____________] *               │
│          (至少8位，包含字母和数字)       │
│                                          │
│  确认密码：[_____________] *             │
│                                          │
│  姓名：  [_____________]                 │
│                                          │
│  头像：  [上传]                          │
│                                          │
│  [取消]              [创建]              │
│                                          │
└─────────────────────────────────────────┘
```

### 验证规则

1. **用户名**：
   - 必填
   - 3-20个字符
   - 只能包含字母、数字、下划线
   - 不能重复

2. **邮箱**：
   - 必填
   - 有效的邮箱格式
   - 不能重复

3. **密码**：
   - 必填
   - 至少8位
   - 包含字母和数字

4. **确认密码**：
   - 必填
   - 与密码一致
